# 🎮 Tic Tac Toe Game (Player vs Player & AI Mode)

An interactive **Tic Tac Toe web application** built using HTML, CSS, and JavaScript.  
This project includes both **Player vs Player** and **Player vs AI** gameplay modes.

## ✨ Features
- 👥 Player vs Player mode
- 🤖 Player vs AI mode
- Intelligent AI move logic (basic level)
- Win detection system
- Draw detection
- Real-time UI updates
- Responsive design
- Clean and minimal interface

- CSS3  
- JavaScript (Vanilla JS)
